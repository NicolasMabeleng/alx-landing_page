About

  • Features

    NM Player is a free and open source cross-platform multimedia player and framework that plays most multimedia files as well as DVDs, Audio CDs, VCDs, and various streaming protocols. Supported Multimedia formats: mp3, m4a, wma, webm, mp4, mkv & more. Music Player Created with Python 3.12.2 using Tkinter library, Pygame 2.5.2 (SDL 2.82.3) and OS module. The tkinter library is used for making the Music Player GUI, os module for the file path, mixer module from pygame library for loading and controlling the music.
